# test-music fixtures

Three short sine-wave MP3s (3 seconds each, A4 / C5 / E5) generated by
`ffmpeg lavfi`. These are NOT loaded by any current E2E test — `07d-05`
plays the production "All Tracks" playlist from `backend/public/music/`
instead.

These exist as a known-good minimal fixture set for future E2E tests
that need predictable short-duration clips (e.g., crossfade timing
tests, gapless boundary tests, playlist transition tests). They're
small enough to commit (~50KB each).

Regenerate with:

```bash
ffmpeg -y -f lavfi -i "sine=frequency=440:duration=3" -ar 44100 -ac 2 -b:a 128k test-track-1.mp3
ffmpeg -y -f lavfi -i "sine=frequency=523:duration=3" -ar 44100 -ac 2 -b:a 128k test-track-2.mp3
ffmpeg -y -f lavfi -i "sine=frequency=659:duration=3" -ar 44100 -ac 2 -b:a 128k test-track-3.mp3
```
