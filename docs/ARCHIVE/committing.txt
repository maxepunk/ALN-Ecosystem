  Alternative: Simpler approach if you want to add everything at once:

  # 1. Commit ALNScanner submodule changes first
  cd ALNScanner
  git add .
  git commit -m "Remove ACK system and add architectural improvements"
  cd ..

  # 2. Add everything in main repo
  git add -A  # This adds all changes including deletions

  # 3. Commit
  git commit -m "your commit message"

  Note: The -A flag in git add -A will:
  - Add all new files
  - Add all modified files
  - Remove all deleted files

  This stages everything shown in git status at once.
